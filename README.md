# 🌙 Nur & Discipline

A motivational website focused on **نور (spiritual light)** and **self-discipline**, inspired by verses from the Quran.

## 📖 About the Project

This platform presents selected Quranic ayahs that encourage reflection, consistency, and personal growth.
The goal is to help users strengthen their discipline and connect with spiritual نور in their daily lives.

## 🚀 Features

* 📜 Quranic verses (ayahs) for motivation
* 💡 Focus on discipline and consistency
* 🌙 Clean and simple user interface
* 📱 Responsive design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 🎯 Purpose

This is a personal project built to combine **technology with meaningful content**, promoting self-improvement and spiritual awareness.
